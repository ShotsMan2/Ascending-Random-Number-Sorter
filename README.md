# Ascending Random Number Sorter

This project implements the **Bubble Sort** algorithm in C to organize a randomly generated dataset.

1.  **Generation:** Creates `N` random integers within $[0, 100]$.
2.  **Sorting (Bubble Sort):**
    * Compares adjacent elements (`arr[j]` vs `arr[j+1]`).
    * **Swap Condition:** If `arr[j] > arr[j+1]`, they swap positions.
    * Repeats until the array is fully sorted from Smallest to Largest.
